# ⚡️ To Do App - SwiftData Edition

Bu proje, **SwiftUI** ve Apple'ın modern veritabanı framework'ü **SwiftData** kullanılarak geliştirilmiş, performans odaklı ve minimalist bir yapılacaklar listesi uygulamasıdır. Kullanıcıların aktivitelerini kalıcı olarak saklamasına, düzenlemesine ve yönetmesine olanak tanır.

## 🚀 Özellikler

  * **SwiftData Entegrasyonu:** Uygulama kapatılsa bile verileriniz güvenli bir şekilde cihazda saklanır.
  * **Tam CRUD Desteği:** Aktivite ekleme (Create), listeleme (Read), düzenleme (Update) ve kaydırarak silme (Delete).
  * **Modern UI/UX:**
      * **Dinamik Arayüz:** `@Query` ile veritabanı değişimlerinde anlık arayüz güncellemesi.
      * **Klavye Yönetimi:** Ekranın boş bir yerine basıldığında otomatik klavye kapatma.
      * **Duyarlı Tasarım:** Liste boş olduğunda kullanıcıyı yönlendiren `ContentUnavailableView`.
      * **Animasyonlar:** `.spring()` animasyonları ile akıcı geçişler.
  * **SF Symbols 5:** En güncel Apple ikon setleri ile görsel bütünlük.

## 🛠 Kullanılan Teknolojiler

  * **Dil:** Swift 5.10+
  * **Framework:** SwiftUI
  * **Veritabanı:** SwiftData
  * **Mimar:** MVVM (Model-View-ViewModel) mantığıyla kurgulanmış deklaratif yapı.

## ⚙️ Kurulum

1.  Bu repoyu bilgisayarınıza clone'layın:
    ```bash
    git clone https://github.com/berkayturan/To-Do-List.git
    ```
2.  `To_Do_List.xcodeproj` dosyasını Xcode ile açın.
3.  Simülatör veya fiziksel bir iOS cihazı seçin.
4.  `Cmd + R` tuşlarına basarak projeyi çalıştırın.
